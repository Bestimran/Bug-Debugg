  • What is the Difference between LocalStorage and sessionStorage?

    • sessionStorage is similar to localStorage ; the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.
    Whenever a document is loaded in a particular tab in the browser, a unique page session gets created and assigned to that particular tab.


  • What is the between block scope and global scope?

    • Function Scope: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function.
    Block Scope: A variable when declared inside the if or switch conditions or inside for or while loops.
